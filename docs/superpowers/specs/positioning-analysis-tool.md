# 낙상 연구 포지셔닝 분석 도구 설계안

## 개요
9편 논문 분석 → 우리 연구(비접촉 센서 + 예측 알고리즘) 포지셔닝 시각화 도구

## 3개 모드
1. **카드 뷰** — 팀 토론용, 논문별 상세 비교 + 필터링
2. **발표 모드** — 교수님 보고용, 스크롤 스토리라인 (포지셔닝 맵 → 갭 분석 → 우리 연구 제안)
3. **대시보드 모드** — 회의 중 한눈에, 맵 + 갭 + 통계 한 화면

## 포지셔닝 맵
- X축: 연구 목적 (예측 ↔ 탐지)
- Y축: 센서 방식 (접촉/데이터 기반 ↔ 비접촉 센서)
- 우리 연구 = 빈 사분면 (비접촉 + 예측/탐지 융합)

## 갭 분석 카테고리
1. 프라이버시 — 카메라 → RF 센서
2. 착용 부담 — IMU/sEMG → 비접촉
3. 실시간성 — EMR/설문 → 실시간 스트리밍
4. 환경 일반화 — 단일 기관 → 다중 환경
