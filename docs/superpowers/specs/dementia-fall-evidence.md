# 치매 환자 낙상 근거 자료

## 1. 왜 치매 환자인가

### 발생률: 일반 노인의 2배
| 구분 | 연간 낙상률 | 재낙상률 |
|---|---|---|
| 일반 노인 (65+) | 약 30% | 25% |
| **치매 환자** | **약 60%** | **40%+** |

- 2026년 한국 치매 환자 **100만 명 돌파** 예상 (유병률 9.17%)
- 혈관성 치매: 초기부터 편측마비·보행장애 동반 → 알츠하이머보다 낙상 위험 더 높음

### 치매 환자만의 낙상 특수성
1. **본인이 위험을 인식 못함** → 시스템이 대신 판단해야
2. **호출벨 사용 불가** → 인지장애로 도움 요청 못함
3. **야간 배회** → 목적 없이 일어나 돌아다님
4. **이중과제 보행 실패** → 걸으면서 다른 생각하면 정상 노인은 보폭 조절하지만 치매 환자는 못함
5. **향정신성 약물 복용** → 어지러움·졸림 → 낙상 위험 증가

## 2. 치매 환자의 보행 특성 (낙상 예측 지표)

### 유아현(2025) 메타분석 결과
| 보행 지표 | 정상 vs MCI | 정상 vs 치매 |
|---|---|---|
| 보행속도 | MD 0.12 감소 | MD 0.21 감소 |
| 케이던스 | MD 9.44 감소 | 분석 불가 (문헌 부족) |
| 활보길이 | MD 0.11 단축 | MD 0.09 단축 |
| 걸음길이 | MD 0.05 단축 | 분석 불가 |

### 국제 연구 추가 지표
| 지표 | 설명 | 낙상 예측 관련성 |
|---|---|---|
| 이중지지시간 변동성 | 양발이 동시에 땅에 닿는 시간의 불규칙도 | 재낙상 핵심 예측 지표 |
| 스텝 시간 변동성(STV) | 걸음 간격의 불규칙도 | 파킨슨·알츠하이머 초기 징후 |
| 보폭 변동성 | 보폭 크기의 불규칙도 | 재낙상 예측 지표 |

**핵심 인사이트:** 치매 환자는 보행 평균값(속도, 보폭)뿐 아니라 **변동성(variability)**이 크게 증가하며, 이 변동성이 낙상 예측에 더 민감한 지표이다.

## 3. 비접촉 센서로 측정 가능한가

### mmWave로 측정 가능 (Zeng 2022 근거)
| 보행 파라미터 | mmWave 측정 정확도 | 방법 |
|---|---|---|
| 스텝 시간 | 96% (ICC 0.93) | micro-Doppler envelope extraction |
| 케이던스 | CVD(Cadence Velocity Diagram) | STFT → 반복 주파수 분석 |
| 절뚝거림/비대칭 보행 | 감지 가능 확인 | envelope 패턴 비대칭 분석 |

### CSI로 측정 가능 (Wang 2016 근거)
| 보행 파라미터 | CSI 측정 정확도 | 방법 |
|---|---|---|
| 보행속도 | 가속도계 대비 오차 11.9ms | spectrogram 주파수 분석 |
| 보행주기(gait cycle) | 50명 검증 | torso contour 자기상관 |
| 보폭 | 상관계수 0.405 (키와 양의 상관) | 속도 × 주기 |
| 다리/몸통 속도 | 실시간 측정 | percentile method |

### mmWave 낙상 탐지 정확도 (Yu et al. 2025 근거)
| 지표 | HOG-LSTM-Attention 성능 | 비고 |
|---|---|---|
| 정확도 | 95.3% | 4종 LSTM 아키텍처 중 최고 |
| 민감도 (Recall) | 98.2% | 실제 낙상의 98.2% 포착 |
| 정밀도 | 92.9% | 오탐 최소화 |
| F1-Score | 95.5% | 균형 잡힌 성능 |

> ※ 원래 이 섹션에 있던 "Gowri 2025" 출처는 실존하지 않는 논문으로 확인되어 Yu et al. 2025 (Applied Sciences)로 교체됨.
> Yu 2025는 낙상 탐지만 다루므로, 침상 움직임/수면/생체신호 근거가 추가로 필요할 경우 별도 논문(Zhang 2025 서베이 등) 확보 필요.

## 4. 우리 연구와의 연결

### 치매 환자 특성 → 센서 감지 → 예측 모델

```
[치매 환자 특성]              [센서 감지]                [예측 출력]
                              
보행속도 점진 감소 ─── mmWave micro-Doppler ───┐
케이던스 불규칙 ──── CSI spectrogram ─────────┤
스텝시간 변동성↑ ─── mmWave envelope ──────────┼→ 낙상 위험도 예측
야간 침상 뒤척임 ─── mmWave 미세움직임 ─────────┤   (즉각 + 추세)
야간 배회 ────────── CSI 벽투과 감지 ────────────┤
기립 시도 ────────── mmWave+CSI 교차검증 ────────┘
```

### 시간 스케일별 예측

| 시간 스케일 | 입력 | 출력 | 대응 |
|---|---|---|---|
| 즉각 (초) | 침상 뒤척임 패턴 → 기립 시도 | "지금 낙상 위험" | 간호사 즉시 알림 |
| 단기 (시간) | 야간 배회 빈도, 보행 불안정도 | "오늘 고위험" | 당직 간호사 주의 |
| 추세 (일/주) | 보행속도·STV 추세 변화 | "낙상 위험도 상승 중" | 의료진 사전 보고 |
