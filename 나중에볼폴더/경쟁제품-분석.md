## 1. VSTAlert (VirtuSense Technologies, 미국)

**한줄 요약**: 적외선 센서 + AI로 침대 이탈 30~65초 전 예측

| 항목 | 내용 |
|---|---|
| 센서 | 적외선(IR) 센서 (카메라 아님, 영상 녹화 없음) |
| 원리 | 수백만 시간 학습 데이터로 "일어나려는 의도" 미세 동작 포착 |
| 정확도 | 감지 98%, 오알람 95% 감소 (자체 발표) |
| 도입 사례 | John Knox Village (80% 낙상 감소), Friendship Village (96% 부상 감소) |
| 가격 | 비공개 (월 구독형 추정) |
| FDA | 미승인 (웰니스 기기 분류) |
| 프라이버시 | 카메라 아님, HIPAA 준수, 영상 없음 |

**한계**:
- 피어리뷰 논문 **0편** (PubMed 검색 결과 없음)
- 침대/의자 이탈만 감지 → 보행 중 낙상은 별도 제품(VSTOne) 필요
- 미국 SNF(전문요양시설) 중심, 해외 진출 미확인
- 센서 기술 상세 비공개 (ToF인지 구조광인지 불명)

**우리와의 차이**:
- VSTAlert = 침대 이탈 "의도" 감지 (사후감지에 가까운 초단기 예측)
- 우리 = 보행 분석 + HRV + 자세로 **당일 위험도 사전 예측** (패러다임이 다름)

> 공식: https://www.virtusense.ai/products/vstalert

## 2. CarePredict Tempo (미국)

**한줄 요약**: 손목 밴드형 웨어러블 + AI 행동 패턴 분석으로 낙상 예측

| 항목 | 내용 |
|---|---|
| 센서 | 모션센서 + 심박 + SpO2 + BLE + RFID + 마이크/스피커 |
| 원리 | 개인별 일상 행동 기준선 학습 → 이상 패턴 감지 → 건강 악화 예측 |
| 정확도 | 감지 98%, 낙상 69% 감소 (6개 시설 연구) |
| 도입 사례 | 전세계 "수백 개" 시설 (공식 표현), TIME 2025 선정 |
| 가격 | 개인용 $499 + 월 $69.99, 시설용 비공개 |
| FDA | 미승인 (웰니스 기기) |

**한계**:
- **착용 필수** → 치매 환자 착용 거부/제거가 근본적 문제
- 피어리뷰 독립 검증 논문 없음
- 24시간 행동 추적 → 프라이버시 우려
- Context Beacon 인프라 설치 필요

**우리와의 차이**:
- CarePredict = 착용형 (치매 환자 착용 거부 문제)
- 우리 = **완전 비접촉** (착용 부담 제로)

> 공식: https://www.carepredict.com/

### Hikvision 레이더 케어
- 60GHz mmWave, 중국 내수 중심

- **사후감지** 추정

### 핵심: mmWave 상용 제품은 전부 "사후감지"
- Vayyar만 "트렌드 분석"으로 제한적 리스크 경고 제공
- 하지만 이것도 "이 사람이 최근 불안정하다" 수준이지, **"오늘 넘어질 확률이 높다"는 당일 예측이 아님**
- **보행 분석 → 당일 위험도 예측하는 mmWave 상용 제품은 현재 0개**

## 경쟁 구도 요약

| 제품 | 방식 | 예측/감지 | 접촉/비접촉 | 프라이버시 | 학술 검증 |
|---|---|---|---|---|---|
| 압력매트 (Stryker) | 압력 센서 | 사후감지 | 비접촉 | O | - |
| VSTAlert | 적외선+AI | 초단기 예측(30~65초) | 비접촉 | O | X (논문 0편) |
| CarePredict | 웨어러블+AI | 행동패턴 예측 | **접촉(착용)** | △ | X |
| Be:Clever (한국) | AI 카메라 | 사후+예측 | 비접촉 | **X (카메라)** | △ |
| mmWave 제품들 | 레이더 | 사후감지 | 비접촉 | O | △ |
| **우리** | mmWave+ToF+CSI | **당일 위험도 예측** | **비접촉** | **O** | 진행 중 |

**우리의 차별점 3줄 요약**:
1. "넘어진 후"도 "넘어지기 직전"도 아닌 **"오늘 아침 위험한가"** 예측
2. 착용 없이 완전 비접촉 (치매 환자 착용 거부 해결)
3. 보행 분석(mmWave) + 자율신경 체크(CSI HRV) + 자세 트리거(ToF) 다중 검증
