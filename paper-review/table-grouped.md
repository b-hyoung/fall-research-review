## 논문 1: Kim et al. (2024), "Fall Detection Using CSI"

### 기존 사례
| 항목 | 내용 |
|------|------|
| 연구 배경 | 노인 낙상 사고 연간 3만건, 사망률 증가 추세 |

### 연구 대상/상황
| 항목 | 내용 |
|------|------|
| 연구 대상 | 65세 이상 노인 |
| 연구 환경 | 요양시설 병실 |
| 연구 목적 | 탐지 |

### 기존 연구 방식
| 항목 | 내용 |
|------|------|
| 사용 센서 | WiFi CSI |
| 알고리즘 | CNN-LSTM |
| 사용 데이터 | 자체 수집 (50명, 6개월) |
| 성능/결과 | 정확도 95.2%, F1 0.93 |
| 한계점 | 단일 환경에서만 검증, 다중 인원 미지원 |

### 차별점
| 항목 | 내용 |
|------|------|
| 우리와의 차이 | 우리는 mmWave 병행으로 다중 인원 커버 가능 |

## 논문 2: Park et al. (2023), "mmWave Radar for Elderly Monitoring"

### 기존 사례
| 항목 | 내용 |
|------|------|
| 연구 배경 | WHO 보고: 낙상은 65세↑ 사망원인 2위 |

### 연구 대상/상황
| 항목 | 내용 |
|------|------|
| 연구 대상 | 뇌졸중 환자 |
| 연구 환경 | 병원 침대 주변 |
| 연구 목적 | 예측 |

### 기존 연구 방식
| 항목 | 내용 |
|------|------|
| 사용 센서 | 60GHz mmWave 레이더 |
| 알고리즘 | Random Forest + SVM |
| 사용 데이터 | 공개 데이터셋 (RadHAR) |
| 성능/결과 | 정확도 91.8% |
| 한계점 | 센서 각도에 민감, 실시간 처리 어려움 |

### 차별점
| 항목 | 내용 |
|------|------|
| 우리와의 차이 | 우리는 CSI 보완으로 사각지대 해소 목표 |
