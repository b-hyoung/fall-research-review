# 논문 비교표 설계 문서

## 목적
- 팀 내부 연구 방향 정리
- 교수님 보고용 자료

## 배경
- CSI / mmWave 기반 비접촉 센서 낙상 연구 방향 탐색 중
- 기존 논문들을 체계적으로 비교 정리할 표 템플릿이 필요
- 교수님 가이드: 센서 종류, 데이터, 알고리즘, 연구 대상/상황, 탐지/예방/예측 구분, 한계점 파악

## 표 칼럼 정의

| # | 칼럼명 | 설명 | 카테고리 |
|---|--------|------|----------|
| 1 | 논문명 | 제목 / 저자 / 년도 | - |
| 2 | 연구 배경 | 왜 낙상 연구가 필요한지 (통계, 근거) | 기존 사례 |
| 3 | 연구 대상 | 누구를 대상으로 (노인, 환자 등) | 연구 대상/상황 |
| 4 | 연구 환경 | 어디서 (병원, 가정, 요양시설 등) | 연구 대상/상황 |
| 5 | 연구 목적 | 탐지 / 예방 / 예측 중 무엇을 다루는지 | 연구 대상/상황 |
| 6 | 사용 센서 | 어떤 센서 (카메라, 웨어러블, CSI, mmWave 등) | 기존 연구 방식 |
| 7 | 알고리즘 | 어떤 ML/DL 모델 사용 | 기존 연구 방식 |
| 8 | 사용 데이터 | 어떤 데이터셋, 데이터 수집 방식 | 기존 연구 방식 |
| 9 | 성능/결과 | 정확도 등 주요 수치 | 기존 연구 방식 |
| 10 | 한계점 | 부족한 점, 남은 문제 | 기존 연구 방식 |
| 11 | 우리와의 차이 | 우리 접근(CSI/mmWave 비접촉)과 다른 점 | 차별점 |

## 산출물

| 파일 | 설명 |
|------|------|
| `paper-review/table-flat.md` | 접근법 A: 논문 1행 = 1줄, 가로 넓은 마크다운 표 |
| `paper-review/table-grouped.md` | 접근법 B: 카테고리별 그룹핑, 세로 구조 마크다운 표 |
| `paper-review/table-visual.html` | 두 접근법 모두 포함된 시각적 HTML (색상 구분, 브라우저에서 열람) |

## 사용 방식
1. 팀원이 논문을 읽고 해당 표의 빈 행을 복사하여 채움
2. 추후 칼럼 추가/수정 가능
3. 예시 데이터 1~2행 포함하여 형식 파악 용이하게 함
