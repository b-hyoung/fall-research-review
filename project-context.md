# 논문 비교표 프로젝트

## 프로젝트 개요
CSI / mmWave 비접촉 센서 기반 **낙상 감지 연구** 논문들을 체계적으로 비교 정리하는 표 제작 프로젝트.

## 목적
- 팀 내부 연구 방향 정리
- 교수님 보고용 자료

## 현재 상태
- 브레인스토밍 완료, 설계 문서 작성됨
- 표 템플릿 3개 파일 생성 완료 (예시 데이터 포함)
- **아직 실제 논문 데이터는 채워넣지 않음** — 팀원들이 논문 읽으면서 채울 예정

## 연구 방향
- **CSI(채널 상태 정보)** 와 **mmWave(밀리미터파 레이더)** 비접촉 센서
- 아직 확정은 아니고 탐색 중

## 접근법 2가지
- **A (table-flat.md)**: 논문 1편 = 1행, 가로로 넓게 비교. 교수님 보고에 적합.
- **B (table-grouped.md)**: 논문별 카테고리 그룹핑, 세로로 깊게 정리. 팀 내부 상세 파악에 적합.

## 교수님 가이드 요약 (디스코드 기반)
- 각 회사/논문에서 어떤 센서를 어떤 방식으로 사용하는가?
- 어떤 환자를 특정해서 연구했는가? 어떤 ML 알고리즘이 좋은지?
- 중요 포인트: 통계 수치, 데이터
- 연구 대상 축소, 알고리즘 유형, 낙상 여부 판단 기준
- 키워드 골라내기, 논문마다 좋은 것들을 골라서 융합하기
- 감지/탐지/예측 구분 필요
- 모든 상황을 다 해결할 수는 없다 — 보편적이되 연구 제약이 필요
